# Изменить адреса электронной почты

Изменяет электронные адреса внешнего контакта. Полностью заменяет список текущих адресов. Новый список не может быть пустым. В новом списке должен быть ровно один адрес с флагом `main=true`.

## Request

```
PUT https://api360.yandex.net/directory/v1/org/{orgId}/external_contacts/{contactId}/emails
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `contactId` | string | Идентификатор внешнего контакта |
| `orgId` | integer | Идентификатор организации |

### Body (application/json)

```json
{
  "emails": [
    {
      "email": "example",
      "type": "example",
      "main": true
    }
  ]
}
```

#### v1ExternalContactEmail

| Параметр | Тип | Описание |
|----------|-----|---------|
| `email` | string | Электронный адрес |
| `main` | boolean | Признак основного адреса: `true` — основной (обязательно ровно один); `false` — дополнительный |
| `type` | string | Тип адреса: `work` — рабочая почта; пусто — личная почта |

## Responses

### 200 OK

Запрос успешно выполнен.

```json
{
  "id": "example"
}
```

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

У пользователя или приложения нет прав на доступ к ресурсу.

### 404 Not Found

Запрашиваемый ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса. Попробуйте повторно отправить запрос позже.
