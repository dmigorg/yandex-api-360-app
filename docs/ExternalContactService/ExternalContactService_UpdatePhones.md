# Изменить номера телефонов

Изменяет телефонные номера внешнего контакта. Полностью заменяет список текущих телефонов. Новый список может быть пустым. В новом списке может быть не более одного телефона с флагом `main=true`.

## Request

```
PUT https://api360.yandex.net/directory/v1/org/{orgId}/external_contacts/{contactId}/phones
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `orgId` | integer | Идентификатор организации |
| `contactId` | string | Идентификатор внешнего контакта |

### Body (application/json)

```json
{
  "phones": [
    {
      "phone": "example",
      "type": "example",
      "main": true
    }
  ]
}
```

#### v1ExternalContactPhone

| Параметр | Тип | Описание |
|----------|-----|---------|
| `phone` | string | Телефон |
| `type` | string | Тип: `work` (рабочий), `mobile` (мобильный), `ip` (IP-телефон), или пусто для прочих |
| `main` | boolean | Основной телефон: `true` или `false` |

## Responses

### 200 OK

Запрос успешно выполнен.

```json
{
  "id": "example"
}
```

| Параметр | Тип | Описание |
|----------|-----|---------|
| `id` | string | Идентификатор внешнего контакта |

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

Нет прав на доступ к ресурсу.

### 404 Not Found

Запрашиваемый ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса.
