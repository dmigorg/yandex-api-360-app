# Внешние контакты

Сервис для работы с внешними контактами (владельцами электронных адресов за пределами вашей организации). Позволяет просматривать, создавать, изменять и удалять контакты.

## Права доступа

- `directory:read_external_contacts` — просмотр
- `directory:write_external_contacts` — просмотр и изменение
