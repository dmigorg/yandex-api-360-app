# Изменить параметры политики паролей

Обновляет параметры политики паролей для домена организации.

## Request

```
PUT https://api360.yandex.net/security/v1/org/{orgId}/domain_passwords
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `orgId` | integer | Идентификатор организации |

### Body (application/json)

```json
{
  "enabled": true,
  "changeFrequency": 0
}
```

| Параметр | Тип | Описание |
|----------|-----|---------|
| `enabled` | boolean | Разрешить пользователям менять пароль |
| `changeFrequency` | integer | Ограничить срок действия пароля (в днях); 0 — без ограничений |

## Responses

### 200 OK

Запрос успешно выполнен.

```json
{
  "enabled": true,
  "changeFrequency": 0
}
```

| Параметр | Тип | Описание |
|----------|-----|---------|
| `enabled` | boolean | Пользователи могут менять пароль |
| `changeFrequency` | integer | Срок действия пароля (в днях) |

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

Недостаточно прав доступа.

### 404 Not Found

Ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса.
