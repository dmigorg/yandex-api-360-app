# Удалить контактную информацию

Удаляет контактную информацию сотрудника, внесённую вручную. Автоматически созданную контактную информацию (с флагом `synthetic`) нельзя удалить.

## Примечание

Требуется разрешение на изменение данных о сотрудниках.

## Request

```
DELETE https://api360.yandex.net/directory/v1/org/{orgId}/users/{userId}/contacts
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `orgId` | integer | Идентификатор организации |
| `userId` | string (uint64) | Идентификатор сотрудника |

## Responses

### 200 OK

Запрос успешно выполнен. Возвращает полный объект пользователя с обновленными данными:

- `id` (string): Идентификатор сотрудника
- `nickname` (string): Логин сотрудника
- `departmentId` (integer): Идентификатор подразделения
- `email` (string): Основной адрес электронной почты
- `name` (v1UserName): ФИО сотрудника
- `contacts` (v1UserContact[]): Список контактов
- `isAdmin` (boolean): Признак администратора
- `isDismissed` (boolean): Статус сотрудника
- `isEnabled` (boolean): Статус аккаунта
- `createdAt` (string): Дата создания
- `updatedAt` (string): Дата изменения

#### v1UserContact

- `type` (string): Тип контакта
- `value` (string): Значение контакта
- `main` (boolean): Основной контакт
- `alias` (boolean): Создан на основе алиаса
- `synthetic` (boolean): Автоматически созданный контакт
- `label` (string): Произвольная метка

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

Недостаточно прав доступа.

### 404 Not Found

Ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса.
