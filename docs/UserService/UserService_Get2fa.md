# Посмотреть статус 2FA

Возвращает информацию о том, настроена ли двухфакторная аутентификация для сотрудника (через сервис v1).

## Примечание

Требуется разрешение на чтение данных о сотрудниках.

## Request

```
GET https://api360.yandex.net/directory/v1/org/{orgId}/users/{userId}/2fa
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `orgId` | integer | Идентификатор организации |
| `userId` | string (uint64) | Идентификатор сотрудника |

## Responses

### 200 OK

Запрос успешно выполнен.

```json
{
  "userId": "example",
  "has2fa": true,
  "hasSecurityPhone": true
}
```

| Параметр | Тип | Описание |
|----------|-----|---------|
| `userId` | string (uint64) | Идентификатор сотрудника |
| `has2fa` | boolean | Включена ли двухфакторная аутентификация: `true` — пользователь самостоятельно настроил 2FA; `false` — администратор включил обязательную 2FA |
| `hasSecurityPhone` | boolean | Настроен ли телефон для безопасности |

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

Недостаточно прав доступа к ресурсу.

### 404 Not Found

Запрашиваемый ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса.
