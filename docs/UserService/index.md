# Сотрудники

## Описание

Сервис предназначен для управления сотрудниками организации, включая их просмотр, создание, изменение и удаление.

## Права доступа

- `directory:read_users` — доступ к просмотру информации
- `directory:write_users` — доступ к просмотру и изменению данных
