# Удалить алиас сотрудника

Удаляет у сотрудника алиас почтового ящика.

## Примечание

Требуется разрешение на изменение данных о сотрудниках.

## Request

```
DELETE https://api360.yandex.net/directory/v1/org/{orgId}/users/{userId}/aliases/{alias}
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `alias` | string | Алиас почтового ящика сотрудника |
| `orgId` | integer | Идентификатор организации |
| `userId` | string (uint64) | Идентификатор сотрудника |

## Responses

### 200 OK

Запрос успешно выполнен.

```json
{
  "alias": "example",
  "removed": true
}
```

| Параметр | Тип | Описание |
|----------|-----|---------|
| `alias` | string | Алиас почтового ящика сотрудника |
| `removed` | boolean | Признак удаления: `true` — удален; `false` — не удален |

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

У пользователя или приложения нет прав на доступ к ресурсу.

### 404 Not Found

Запрашиваемый ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса. Попробуйте повторно отправить запрос через некоторое время.
