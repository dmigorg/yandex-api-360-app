# Посмотреть статус персональной 2FA

Получает информацию о включении двухфакторной аутентификации для отдельного сотрудника.

## Доступность

Доступен только в тарифах «Корпоративный + Безопасность» и «Корпоративный Максимум».

Метод работает при следующих условиях:
- Организация использует сервис управления 2FA (v2) с возможностью персональной настройки
- Режим двухфакторной аутентификации установлен как `scope: per_user`

## Request

```
GET https://api360.yandex.net/directory/v1/org/{orgId}/users/{userId}/domain_2fa
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `orgId` | integer | Идентификатор организации |
| `userId` | string (uint64) | Идентификатор сотрудника |

## Responses

### 200 OK

Запрос успешно выполнен.

```json
{
  "id": "example",
  "is2faEnabled": true
}
```

| Параметр | Тип | Описание |
|----------|-----|---------|
| `id` | string (uint64) | Идентификатор сотрудника |
| `is2faEnabled` | boolean | Статус 2FA: `true` — включена; `false` — отключена |

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

Недостаточно прав доступа.

### 404 Not Found

Ресурс не найден.

### 500 Internal Server Error

Ошибка сервиса.
