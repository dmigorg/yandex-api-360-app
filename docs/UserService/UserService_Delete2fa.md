# Сбросить телефон для 2FA

Сбрасывает телефон для двухфакторной аутентификации у сотрудника организации.

## Важная информация

Если пользователь не добавил номер телефона для двухфакторной аутентификации, запрос вернет ошибку `400`. Проверить наличие телефона можно через поле `hasSecurityPhone` в ответе на запрос статуса 2FA.

## Примечание

Требуется разрешение на изменение данных о сотрудниках.

## Request

```
DELETE https://api360.yandex.net/directory/v1/org/{orgId}/users/{userId}/2fa
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `orgId` | integer | Идентификатор организации |
| `userId` | string (uint64) | Идентификатор сотрудника |

## Responses

### 200 OK

Запрос успешно выполнен.

**Body:** `{}`

### 400 Bad Request

Некорректный запрос (например, если у пользователя нет телефона для 2FA).

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

Недостаточно прав доступа к ресурсу.

### 404 Not Found

Запрашиваемый ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса. Рекомендуется повторить запрос позже.
