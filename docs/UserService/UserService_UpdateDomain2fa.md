# Изменить статус персональной 2FA

Обновляет статус персональной двухфакторной аутентификации для отдельного сотрудника.

## Доступность

Доступен только в тарифах «Корпоративный + Безопасность» и «Корпоративный Максимум».

Метод применим при следующих условиях:
- Организация использует сервис управления 2FA (v2) с возможностью персональной настройки
- Режим двухфакторной аутентификации установлен как `scope: per_user`

## Примечание

Требуется разрешение на изменение данных о сотрудниках.

## Request

```
PATCH https://api360.yandex.net/directory/v1/org/{orgId}/users/{userId}/domain_2fa
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `orgId` | integer | Идентификатор организации |
| `userId` | string (uint64) | Идентификатор сотрудника |

### Query-параметры

| Параметр | Тип | Описание |
|----------|-----|---------|
| `is2faEnabled` | boolean | `true` — включить персональную 2FA; `false` — отключить |

## Responses

### 200 OK

Запрос успешно выполнен.

**Body:** `{}`

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

У пользователя или приложения нет прав на доступ к ресурсу.

### 404 Not Found

Запрашиваемый ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса. Повторите запрос позже.
