# Изменить статус автоматического сбора контактов

Управляет функцией автоматического создания адресной книги сотрудника на основе адресов отправляемых писем.

## Примечание

Для выполнения запроса приложение должно иметь разрешение на управление параметрами почтового ящика пользователя при настройке на OAuth-сервере Яндекса.

## Request

```
POST https://api360.yandex.net/admin/v1/org/{orgId}/mail/users/{userId}/settings/address_book
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `orgId` | integer | Идентификатор организации |
| `userId` | string (uint64) | Идентификатор пользователя |

### Body (application/json)

```json
{
  "collectAddresses": true
}
```

| Параметр | Тип | Описание |
|----------|-----|---------|
| `collectAddresses` | boolean | Автоматически собирать контакты |

## Responses

### 200 OK

Запрос успешно выполнен.

```json
{
  "collectAddresses": true
}
```

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

Недостаточно прав для доступа к ресурсу.

### 404 Not Found

Запрашиваемый ресурс отсутствует.

### 500 Internal Server Error

Ошибка сервера. Рекомендуется повторить попытку позже.
