# Удалить правило автоответа или пересылки

Удаляет правило автоответа или пересылки писем, настроенное для сотрудника.

## Примечание

Для выполнения запроса требуется разрешение на управление настройками почты пользователя при настройке OAuth-приложения.

## Request

```
DELETE https://api360.yandex.net/admin/v1/org/{orgId}/mail/users/{userId}/settings/user_rules/{ruleId}
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `orgId` | integer | Идентификатор организации |
| `userId` | string (uint64) | Идентификатор пользователя |
| `ruleId` | integer | Идентификатор правила |

## Responses

### 200 OK

Запрос успешно выполнен.

**Body:** `{}`

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

Недостаточно прав доступа.

### 404 Not Found

Ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса.
