# Просмотреть правила автоответа и пересылки

Возвращает правила автоответа и пересылки писем для сотрудника.

## Примечание

Приложению требуется разрешение на чтение настроек почты пользователя при настройке на OAuth-сервере Яндекса.

## Request

```
GET https://api360.yandex.net/admin/v1/org/{orgId}/mail/users/{userId}/settings/user_rules
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `orgId` | integer | Идентификатор организации |
| `userId` | string (uint64) | Идентификатор пользователя |

## Responses

### 200 OK

Запрос успешно выполнен.

**Response Body:**

```json
{
  "autoreplies": [
    {
      "ruleId": 0,
      "ruleName": "example",
      "text": "example"
    }
  ],
  "forwards": [
    {
      "ruleId": 0,
      "ruleName": "example",
      "address": "example",
      "withStore": true
    }
  ]
}
```

**Правило автоответа:**

| Параметр | Тип | Описание |
|----------|-----|---------|
| `ruleId` | integer | Идентификатор правила |
| `ruleName` | string | Название правила |
| `text` | string | Текст автоответа |

**Правило пересылки:**

| Параметр | Тип | Описание |
|----------|-----|---------|
| `ruleId` | integer | Идентификатор правила |
| `ruleName` | string | Название правила |
| `address` | string | Адрес электронной почты получателя |
| `withStore` | boolean | Сохранять ли копию в исходном ящике |

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

Недостаточно прав доступа.

### 404 Not Found

Ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса. Рекомендуется повторить запрос.
