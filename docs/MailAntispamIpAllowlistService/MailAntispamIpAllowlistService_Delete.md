# Удалить список антиспама

Удаляет список разрешенных IP-адресов и CIDR-подсетей.

## Примечание

Чтобы выполнить запрос, приложению требуется разрешение на управление настройками антиспама. Убедитесь, что вы включили его при настройке приложения на OAuth-сервере Яндекса.

## Request

```
DELETE https://api360.yandex.net/admin/v1/org/{orgId}/mail/antispam/allowlist/ips
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `orgId` | integer | Идентификатор организации |

## Responses

### 200 OK

Запрос успешно выполнен.

**Body:** `{}`

### 400 Bad Request

Некорректный запрос.

```json
{
  "code": 0,
  "message": "example",
  "details": [
    {
      "@type": "example"
    }
  ]
}
```

| Параметр | Тип | Описание |
|----------|-----|---------|
| `code` | integer | Код ошибки |
| `message` | string | Описание ошибки |
| `details` | protobufAny[] | Дополнительные сведения об ошибке |

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

У пользователя или приложения нет прав на доступ к ресурсу, запрос отклонен.

### 404 Not Found

Запрашиваемый ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса. Попробуйте повторно отправить запрос через некоторое время.
