# Организации

## Описание

Сервис предназначен для просмотра организаций пользователя в системе Яндекс 360 API.

## Права доступа

- `directory:read_organization` — позволяет просматривать информацию об организациях
- `directory:write_organization` — дает возможность просматривать и изменять данные организаций
