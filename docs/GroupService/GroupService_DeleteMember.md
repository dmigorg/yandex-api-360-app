# Удалить участника из группы

Удаляет конкретного участника из группы: сотрудника организации, подразделение или вложенную группу.

## Примечание

Требуется разрешение на изменение данных о группах.

## Request

```
DELETE https://api360.yandex.net/directory/v1/org/{orgId}/groups/{groupId}/members/{type}/{id}
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `orgId` | integer | Идентификатор организации |
| `groupId` | integer | Идентификатор группы |
| `type` | string | Тип участника: `user`, `group`, `department` |
| `id` | string (uint64) | Идентификатор участника группы |

## Responses

### 200 OK

Запрос успешно выполнен.

```json
{
  "id": "example",
  "type": "example",
  "deleted": true
}
```

| Параметр | Тип | Описание |
|----------|-----|---------|
| `id` | string (uint64) | Идентификатор участника группы |
| `type` | string | Тип участника группы |
| `deleted` | boolean | Признак удаления: `true` — удален; `false` — не удален |

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

У пользователя или приложения нет прав на доступ к ресурсу, запрос отклонен.

### 404 Not Found

Запрашиваемый ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса. Попробуйте повторно отправить запрос через некоторое время.
