# Просмотреть список участников группы

Возвращает список участников группы: сотрудников, подразделений или других групп.

## Примечание

Требуется разрешение на чтение данных о группах.

## Request

```
GET https://api360.yandex.net/directory/v1/org/{orgId}/groups/{groupId}/members
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `groupId` | integer | Идентификатор группы |
| `orgId` | integer | Идентификатор организации |

## Responses

### 200 OK

Запрос успешно выполнен.

```json
{
  "departments": [...],
  "groups": [...],
  "users": [...]
}
```

| Параметр | Тип | Описание |
|----------|-----|---------|
| `departments` | v1ShortDepartment[] | Информация о подразделениях |
| `groups` | v1ShortGroup[] | Информация о группах |
| `users` | v1ShortUser[] | Информация о сотрудниках |

#### v1ShortDepartment

| Параметр | Тип | Описание |
|----------|-----|---------|
| `id` | integer | Идентификатор подразделения |
| `name` | string | Название подразделения |
| `membersCount` | integer | Количество сотрудников (включая вложенные подразделения) |

#### v1ShortGroup

| Параметр | Тип | Описание |
|----------|-----|---------|
| `id` | integer | Идентификатор группы |
| `name` | string | Название группы |
| `membersCount` | integer | Количество участников группы |

#### v1ShortUser

| Параметр | Тип | Описание |
|----------|-----|---------|
| `id` | string | Идентификатор сотрудника |
| `nickname` | string | Логин сотрудника |
| `departmentId` | integer | Идентификатор подразделения |
| `email` | string | Основной адрес электронной почты |
| `name` | v1UserName | Полное имя сотрудника |
| `gender` | string | Пол сотрудника |
| `position` | string | Должность сотрудника |
| `avatarId` | string | Идентификатор портрета |

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

Недостаточно прав доступа к ресурсу.

### 404 Not Found

Запрашиваемый ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса.
