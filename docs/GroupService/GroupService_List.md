# Просмотреть список групп

Возвращает список групп с постраничной навигацией.

## Примечание

Требуется разрешение на чтение данных о группах.

## Request

```
GET https://api360.yandex.net/directory/v1/org/{orgId}/groups
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `orgId` | integer | Идентификатор организации |

### Query-параметры

| Параметр | Тип | Описание | По умолчанию |
|----------|-----|---------|--------------|
| `page` | integer | Номер страницы ответа | `1` |
| `perPage` | integer | Количество групп на одной странице | `10` |

## Responses

### 200 OK

Запрос успешно выполнен.

**Response Body:**

```json
{
  "groups": [...],
  "page": 0,
  "pages": 0,
  "perPage": 0,
  "total": 0
}
```

| Параметр | Тип | Описание |
|----------|-----|---------|
| `groups` | v1Group[] | Список групп |
| `page` | integer | Номер страницы ответа |
| `pages` | integer | Количество страниц ответа |
| `perPage` | integer | Количество групп на одной странице |
| `total` | integer | Общее количество групп |

#### v1Group

| Параметр | Тип | Описание |
|----------|-----|---------|
| `id` | integer | Идентификатор группы |
| `name` | string | Название группы |
| `type` | string | Тип группы: `generic`, `organization_admin`, `robots`, `organization_deputy_admin` |
| `description` | string | Описание группы |
| `membersCount` | integer | Количество участников |
| `label` | string | Имя почтовой рассылки |
| `email` | string | Адрес почтовой рассылки |
| `emailId` | string (uint64) | Идентификатор почтовой рассылки |
| `aliases` | string[] | Алиасы почтовых ящиков |
| `externalId` | string | Произвольный внешний идентификатор |
| `removed` | boolean | Признак удаленной группы |
| `members` | v1GroupMember[] | Участники группы |
| `memberOf` | integer[] | Идентификаторы групп, в которые входит эта группа |
| `createdAt` | string (date-time) | Дата и время создания |

#### v1GroupMember

| Параметр | Тип | Описание |
|----------|-----|---------|
| `id` | string (uint64) | Идентификатор участника |
| `type` | string | Тип участника: `user`, `group`, `department` |

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

Недостаточно прав доступа.

### 404 Not Found

Запрашиваемый ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса.
