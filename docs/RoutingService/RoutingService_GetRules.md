# Просмотреть список правил обработки писем

Возвращает список правил обработки писем.

## Примечание

Чтобы выполнить запрос, приложению требуется разрешение на чтение правил обработки почты для домена.

## Request

```
GET https://api360.yandex.net/admin/v1/org/{orgId}/mail/routing/rules
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `orgId` | integer | Идентификатор организации |

## Responses

### 200 OK

Запрос успешно выполнен.

**Response Body:**

```json
{
  "rules": [
    {
      "terminal": true,
      "condition": null,
      "actions": [
        null
      ]
    }
  ]
}
```

| Параметр | Тип | Описание |
|----------|-----|---------|
| `rules` | adminv1Rule[] | Список правил обработки писем |
| `rules[].actions` | unknown[] | JSON-описание действий при срабатывании правила |
| `rules[].condition` | unknown | JSON-описание условия соответствия письма правилу |
| `rules[].terminal` | boolean | Флаг прекращения применения последующих правил при срабатывании данного |

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

У пользователя или приложения нет прав на доступ к ресурсу, запрос отклонен.

### 404 Not Found

Запрашиваемый ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса. Попробуйте повторно отправить запрос через некоторое время.
