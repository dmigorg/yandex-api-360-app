# Задать список правил обработки писем

Создает или обновляет список правил обработки писем.

## Примечание

Приложению требуется разрешение на управление правилами обработки почты для домена при настройке на OAuth-сервере Яндекса.

## Request

```
PUT https://api360.yandex.net/admin/v1/org/{orgId}/mail/routing/rules
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `orgId` | integer | Идентификатор организации |

### Body (application/json)

```json
{
  "rules": [
    {
      "condition": {
        "address:from": "spam@example.com"
      },
      "actions": [
        {
          "action": "drop"
        }
      ],
      "scope": {
        "direction": "inbound"
      },
      "terminal": true
    }
  ]
}
```

| Параметр | Тип | Описание |
|----------|-----|---------|
| `rules` | v1RuleParams[] | Список правил обработки писем |

#### v1RuleParams

| Параметр | Тип | Описание |
|----------|-----|---------|
| `actions` | v1RuleActionParams[] | Действия при срабатывании правила |
| `condition` | unknown | Условие проверки соответствия письма правилу |
| `scope` | v1RuleScopeParams | Область применения правила |
| `terminal` | boolean | Прекратить применение последующих правил при срабатывании данного |

#### v1RuleActionParams

| Параметр | Тип | Описание |
|----------|-----|---------|
| `action` | string | Действие над письмом: `drop` — не доставлять; `forward` — переслать |
| `data` | v1RuleActionData | Дополнительные данные для действия |

#### v1RuleActionData

| Параметр | Тип | Описание |
|----------|-----|---------|
| `email` | string | Адрес для пересылки письма (используется при `action: forward`) |

#### v1RuleScopeParams

| Параметр | Тип | Описание |
|----------|-----|---------|
| `direction` | string | Область применения: `inbound` — входящая почта; `outbound` — исходящая почта |

## Responses

### 200 OK

Запрос успешно выполнен.

**Body:** `{}`

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

У пользователя или приложения нет прав на доступ к ресурсу.

### 404 Not Found

Запрашиваемый ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса. Попробуйте повторно отправить запрос через некоторое время.
