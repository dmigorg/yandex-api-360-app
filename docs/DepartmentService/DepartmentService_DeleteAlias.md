# Удалить алиас у подразделения

Удаляет алиас почтовой рассылки подразделения.

## Примечание

Требуется разрешение на изменение данных о подразделениях.

## Request

```
DELETE https://api360.yandex.net/directory/v1/org/{orgId}/departments/{departmentId}/aliases/{alias}
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `alias` | string | Алиас почтовой рассылки подразделения |
| `departmentId` | integer | Идентификатор подразделения |
| `orgId` | integer | Идентификатор организации |

## Responses

### 200 OK

Запрос успешно выполнен.

```json
{
  "alias": "example",
  "removed": true
}
```

| Параметр | Тип | Описание |
|----------|-----|---------|
| `alias` | string | Алиас почтовой рассылки подразделения |
| `removed` | boolean | Признак удаления: `true` — удален; `false` — не удален |

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

У пользователя или приложения нет прав на доступ к ресурсу, запрос отклонен.

### 404 Not Found

Запрашиваемый ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса. Попробуйте повторно отправить запрос через некоторое время.
