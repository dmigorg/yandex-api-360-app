# Посмотреть информацию о подразделении

Возвращает информацию об одном подразделении.

## Примечание

Требуется разрешение на чтение данных о подразделениях.

## Request

```
GET https://api360.yandex.net/directory/v1/org/{orgId}/departments/{departmentId}
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `departmentId` | integer | Идентификатор подразделения |
| `orgId` | integer | Идентификатор организации |

## Responses

### 200 OK

Запрос успешно выполнен.

```json
{
  "id": 0,
  "name": "example",
  "parentId": 0,
  "description": "example",
  "createdAt": "2025-01-01T00:00:00Z",
  "externalId": "example",
  "label": "example",
  "email": "example",
  "emailId": "example",
  "membersCount": 0,
  "aliases": [
    "example"
  ]
}
```

| Параметр | Тип | Описание |
|----------|-----|---------|
| `aliases` | string[] | Алиасы почтовых рассылок |
| `createdAt` | string (date-time) | Дата и время создания подразделения |
| `description` | string | Описание подразделения |
| `email` | string | Адрес почтовой рассылки подразделения |
| `emailId` | string (uint64) | Идентификатор почтовой рассылки подразделения |
| `externalId` | string | Произвольный внешний идентификатор подразделения |
| `id` | integer | Идентификатор подразделения |
| `label` | string | Имя почтовой рассылки подразделения |
| `membersCount` | integer | Количество сотрудников подразделения включая вложенные подразделения |
| `name` | string | Название подразделения |
| `parentId` | integer | Идентификатор родительского подразделения |

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

У пользователя или приложения нет прав на доступ к ресурсу, запрос отклонен.

### 404 Not Found

Запрашиваемый ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса. Попробуйте повторно отправить запрос через некоторое время.
