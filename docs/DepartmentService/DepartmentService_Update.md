# Изменить параметры подразделения

Изменяет информацию о подразделении. Изменяются значения только тех параметров, которые были переданы в запросе.

## Примечание

Требуется разрешение на изменение данных о подразделениях.

## Request

```
PATCH https://api360.yandex.net/directory/v1/org/{orgId}/departments/{departmentId}
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `departmentId` | integer | Идентификатор подразделения |
| `orgId` | integer | Идентификатор организации |

### Body (application/json)

| Параметр | Тип | Описание |
|----------|-----|---------|
| `name` | string | Название подразделения |
| `parentId` | integer | Идентификатор родительского подразделения |
| `description` | string | Описание подразделения |
| `externalId` | string | Произвольный внешний идентификатор подразделения |
| `label` | string | Имя почтовой рассылки подразделения (например, `new-department` для адреса `new-department@ваш-домен.ru`) |

## Responses

### 200 OK

Запрос успешно выполнен.

| Параметр | Тип | Описание |
|----------|-----|---------|
| `id` | integer | Идентификатор подразделения |
| `name` | string | Название подразделения |
| `parentId` | integer | Идентификатор родительского подразделения |
| `description` | string | Описание подразделения |
| `createdAt` | string (date-time) | Дата и время создания подразделения |
| `externalId` | string | Произвольный внешний идентификатор подразделения |
| `label` | string | Имя почтовой рассылки подразделения |
| `email` | string | Адрес почтовой рассылки подразделения |
| `emailId` | string (uint64) | Идентификатор почтовой рассылки подразделения |
| `membersCount` | integer | Количество сотрудников подразделения с учетом вложенных подразделений |
| `aliases` | string[] | Алиасы почтовых рассылок |

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

У пользователя или приложения нет прав на доступ к ресурсу.

### 404 Not Found

Запрашиваемый ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса. Попробуйте повторно отправить запрос позже.
