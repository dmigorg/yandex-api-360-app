# Удалить DNS-запись

Удаляет DNS-запись для домена по её идентификатору.

## Примечание

Требуется разрешение на управление DNS (чтение и запись).

## Request

```
DELETE https://api360.yandex.net/directory/v1/org/{orgId}/domains/{domain}/dns/{recordId}
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `domain` | string | Полное доменное имя (например `example.com`). Для кириллических доменов используйте кодировку Punycode |
| `orgId` | integer | Идентификатор организации |
| `recordId` | integer | Идентификатор записи |

## Responses

### 200 OK

Запрос успешно выполнен.

**Body:** `{}`

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

У пользователя или приложения нет прав на доступ к ресурсу, запрос отклонен.

### 404 Not Found

Запрашиваемый ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса. Попробуйте повторно отправить запрос через некоторое время.

Все ошибочные ответы содержат поля `code`, `message` и `details`.
