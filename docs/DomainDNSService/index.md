# DNS

С помощью запросов на управление DNS можно редактировать DNS-записи для доменов, делегированных на серверы Яндекса.

Если домен ещё не делегирован, эти запросы позволяют добавить необходимые DNS-записи на серверы Яндекса для последующей делегации.

## Права доступа

- `directory:manage_dns` — администрирование DNS (доступ на чтение и запись)
