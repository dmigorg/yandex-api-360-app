# Безопасность: запрет на авторизацию сотрудников во внешних OAuth-сервисах

## Обзор

Сервис позволяет организациям устанавливать или снимать запрет на авторизацию сотрудников во внешних OAuth-сервисах. По умолчанию такой запрет не установлен.

Функция позволяет предотвратить использование сотрудниками рабочих аккаунтов для доступа к сторонним приложениям и сервисам, что повышает безопасность организации.

## Разрешения на использование сервиса

- `ya360_security:domain_settings_read` — чтение конфигурации безопасности организации
- `ya360_security:domain_settings_write` — изменение настроек безопасности организации
