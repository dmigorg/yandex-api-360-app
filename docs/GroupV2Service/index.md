# Группы (v2)

## Обзор

Дополнительный сервис для работы с группами, которые могут содержать внешние контакты.

Методы v2 предназначены для операций с участниками группы: добавление, просмотр и удаление. Для работы с самими группами (их создание, удаление, просмотр списка) и параметрами, а также управлением руководителями, применяются стандартные методы, даже если группы содержат внешние контакты.

## Права доступа

- `directory:read_groups` — для просмотра
- `directory:write_groups` — для просмотра и изменения
