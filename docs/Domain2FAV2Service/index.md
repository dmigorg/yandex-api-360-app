# Безопасность: двухфакторная аутентификация с возможностью персональной настройки (v2)

## Обзор

Сервис позволяет управлять обязательной двухфакторной аутентификацией (2FA) с гибкими настройками. Организации могут применять настройки 2FA для всей организации в режиме `per_domain` или выборочно для отдельных сотрудников в режиме `per_user`.

## Доступность

Функция доступна в тарифах для крупных организаций **Корпоративный + Безопасность** и **Корпоративный Максимум**.

## Переключение режима: per_domain → per_user

При переключении с режима `per_domain` на `per_user`:

- Пользователи, которые самостоятельно включили 2FA ранее, сохраняют свои настройки
- Для других пользователей 2FA будет отключена

В режиме `per_user` сотрудники могут самостоятельно включать или отключать 2FA в настройках безопасности Яндекс ID.

## Разрешения на использование сервиса

- `ya360_security:domain_2fa_write` — управление обязательной двухфакторной аутентификацией для пользователей
