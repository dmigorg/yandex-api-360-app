# Безопасность: двухфакторная аутентификация (v1)

## Обзор

Сервис позволяет организациям внедрять обязательную двухфакторную аутентификацию (2FA) для всех аккаунтов пользователей на домене. API предоставляет методы для управления настройками 2FA на уровне организации.

## Ключевые возможности

Сервис позволяет администраторам включать или отключать двухфакторную аутентификацию одновременно для всех пользователей с аккаунтами на домене организации. Для управления 2FA на уровне отдельных пользователей используйте сервис v2 (доступен только на определённых тарифах).

## Разрешения на использование сервиса

- `ya360_security:domain_2fa_write` — управление обязательной 2FA для пользователей
