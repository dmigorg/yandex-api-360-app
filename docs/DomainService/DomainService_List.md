# Посмотреть список доменов

Возвращает список доменов организации.

## Примечание

Требуется разрешение на чтение данных о доменах.

## Request

```
GET https://api360.yandex.net/directory/v1/org/{orgId}/domains
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `orgId` | integer | Идентификатор организации |

### Query-параметры

| Параметр | Тип | Описание | По умолчанию |
|----------|-----|---------|--------------|
| `page` | integer | Номер страницы ответа | `1` |
| `perPage` | integer | Количество доменов на одной странице. Максимум — `10` | `10` |

## Responses

### 200 OK

Запрос успешно выполнен.

**Response Body:**

```json
{
  "domains": [
    {
      "name": "example",
      "country": "example",
      "mx": true,
      "delegated": true,
      "master": true,
      "verified": true,
      "status": {
        "name": "example",
        "lastCheck": "2025-01-01T00:00:00Z",
        "lastAdded": "2025-01-01T00:00:00Z",
        "spf": {
          "match": true,
          "value": "example"
        }
      }
    }
  ],
  "page": 0,
  "pages": 0,
  "perPage": 0,
  "total": 0
}
```

| Параметр | Тип | Описание |
|----------|-----|---------|
| `domains` | v1Domain[] | Список доменов |
| `page` | integer | Номер страницы ответа |
| `pages` | integer | Количество страниц ответа |
| `perPage` | integer | Количество доменов на одной странице |
| `total` | integer | Общее количество доменов |

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

У пользователя или приложения нет прав на доступ к ресурсу.

### 404 Not Found

Запрашиваемый ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса. Попробуйте повторно отправить запрос позже.
