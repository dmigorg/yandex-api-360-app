# Удалить домен

Позволяет удалить домен. Вы можете удалить любой домен, кроме технического.

## Примечание

Требуется разрешение на запись данных о доменах.

## Request

```
DELETE https://api360.yandex.net/directory/v1/org/{orgId}/domains/{domain}
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `domain` | string | Полное имя домена |
| `orgId` | integer | Идентификатор организации |

## Responses

### 200 OK

Запрос успешно выполнен.

**Body:** `{}`

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

У пользователя или приложения нет прав на доступ к ресурсу.

### 404 Not Found

Запрашиваемый ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса. Попробуйте повторно отправить запрос через некоторое время.
