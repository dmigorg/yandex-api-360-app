# Получить статус DKIM для домена

Возвращает статус DKIM подписи для указанного домена.

## Примечание

Требуется разрешение на чтение данных о доменах.

## Request

```
GET https://api360.yandex.net/directory/v1/org/{orgId}/domains/{domain}/dkim
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `orgId` | integer | Идентификатор организации |
| `domain` | string | Полное имя домена (например `example.com`). Для кириллических доменов используйте кодировку Punycode |

## Responses

### 200 OK

Запрос успешно выполнен.

```json
{
  "enabled": true,
  "publicKey": "example"
}
```

| Параметр | Тип | Описание |
|----------|-----|---------|
| `enabled` | boolean | Статус DKIM: `true` — подпись включена; `false` — выключена |
| `publicKey` | string | Значение публичного ключа |

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

Недостаточно прав доступа.

### 404 Not Found

Ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса.
