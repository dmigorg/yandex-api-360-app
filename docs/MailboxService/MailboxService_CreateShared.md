# Создать общий ящик

Создает общий ящик в организации.

В ответе на запрос приходит поле `resourceId` — идентификатор созданного общего почтового ящика.

## Примечание

Чтобы выполнить запрос, приложению требуется разрешение на управление правами доступа к почтовым ящикам. Убедитесь, что вы включили его при настройке приложения на OAuth-сервере Яндекса.

## Request

```
PUT https://api360.yandex.net/admin/v1/org/{orgId}/mailboxes/shared
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `orgId` | integer | Идентификатор организации |

### Body (application/json)

```json
{
  "email": "example",
  "name": "example",
  "description": "example"
}
```

| Параметр | Тип | Описание |
|----------|-----|---------|
| `email` | string | Адрес электронной почты общего ящика |
| `name` | string | Имя общего ящика |
| `description` | string | Описание |

## Responses

### 200 OK

Запрос успешно выполнен.

**Body:**

```json
{
  "resourceId": "example"
}
```

| Параметр | Тип | Описание |
|----------|-----|---------|
| `resourceId` | string (uint64) | Идентификатор созданного почтового ящика |

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

У пользователя или приложения нет прав на доступ к ресурсу, запрос отклонен.

### 404 Not Found

Запрашиваемый ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса. Попробуйте повторно отправить запрос через некоторое время.
