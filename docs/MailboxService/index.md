# Общие и делегированные ящики

Сервис для управления доступом пользователей к общим ящикам и к делегированным ящикам других сотрудников.

Возможность использования общих и делегированных ящиков есть не во всех тарифах Яндекс 360 для бизнеса. Описание тарифов приведено в Справке по тарифам и оплате.

## Разрешения на использование сервиса

- `ya360_admin:mail_write_shared_mailbox_inventory` — управление правами доступа к почтовым ящикам
- `ya360_admin:mail_read_shared_mailbox_inventory` — чтение информации о правах доступа к почтовым ящикам

## Ограничения для общих и делегированных ящиков

### Лимиты

| Параметр | Делегированные ящики | Общие ящики |
|----------|---------------------|------------|
| Количество ящиков в организации | 2500 | 2500 |
| Количество ящиков, к которым может иметь доступ один пользователь | 40 (общий лимит) | |
| Количество пользователей, которые могут иметь доступ к одному ящику | 40 | 40 |

### Операционные ограничения

Групповые операции на предоставление доступов пока не поддерживаются. За один запрос можно предоставить, изменить или удалить право доступа только для одного сотрудника к одному почтовому ящику. Можно одновременно направить несколько таких запросов.
