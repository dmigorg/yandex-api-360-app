# Изменить права доступа к ящику

Предоставляет или изменяет права доступа сотрудника к делегированному или общему почтовому ящику. Чтобы ящик другого сотрудника стал делегированным и к нему можно было получить доступ, сначала включите возможность его делегирования.

Действие асинхронное: можно отправлять следующий запрос, не дожидаясь выполнения предыдущего. Возвращает идентификатор задачи, по которому можно проверить состояние ее выполнения.

## Примечание

Чтобы выполнить запрос, приложению требуется разрешение на управление правами доступа к почтовым ящикам.

## Request

```
POST https://api360.yandex.net/admin/v1/org/{orgId}/mailboxes/set/{resourceId}
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `orgId` | integer | Идентификатор организации |
| `resourceId` | string (uint64) | Идентификатор почтового ящика |

### Query-параметры

| Параметр | Тип | Описание |
|----------|-----|---------|
| `actorId` | string (uint64) | Идентификатор сотрудника, для которого настраивается доступ |
| `notify` | string | Кому отправить письмо-уведомление: `all`, `delegates`, `none`. По умолчанию: `all` |

### Body (application/json)

```json
{
  "roles": [
    "example"
  ]
}
```

| Параметр | Тип | Описание |
|----------|-----|---------|
| `roles` | string[] | Список ролей для назначения сотруднику |

**Возможные значения ролей:**

- `shared_mailbox_reader` — просмотр ящика
- `shared_mailbox_editor` — редактирование ящика
- `shared_mailbox_admin` — управление ящиком
- `shared_mailbox_imap_admin` — управление ящиком в IMAP-клиенте
- `shared_mailbox_sender` — отправка писем
- `shared_mailbox_half_sender` — ограниченная отправка писем
- `shared_mailbox_owner` — полные права на ящик

## Responses

### 200 OK

Запрос успешно выполнен.

```json
{
  "taskId": "example"
}
```

| Параметр | Тип | Описание |
|----------|-----|---------|
| `taskId` | string | Идентификатор задачи для проверки состояния |

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

У пользователя или приложения нет прав доступа к ресурсу.

### 404 Not Found

Запрашиваемый ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса. Требуется повторная отправка запроса.
