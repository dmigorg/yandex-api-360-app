# Посмотреть список общих ящиков

Возвращает список общих почтовых ящиков в организации.

## Примечание

Чтобы выполнить запрос, приложению требуется разрешение на чтение информации о правах доступа к почтовым ящикам. Убедитесь, что вы включили его при настройке приложения на OAuth-сервере Яндекса.

## Request

```
GET https://api360.yandex.net/admin/v1/org/{orgId}/mailboxes/shared
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `orgId` | integer | Идентификатор организации |

### Query-параметры

| Параметр | Тип | Описание |
|----------|-----|---------|
| `page` | integer | Номер страницы ответа. По умолчанию: `1` |
| `perPage` | integer | Количество записей на одной странице ответа. По умолчанию: `10` |

## Responses

### 200 OK

Запрос успешно выполнен.

**Response Body:**

```json
{
  "resources": [
    {
      "resourceId": "example",
      "count": 0
    }
  ],
  "page": 0,
  "perPage": 0,
  "total": 0
}
```

| Параметр | Тип | Описание |
|----------|-----|---------|
| `page` | integer | Номер страницы ответа |
| `perPage` | integer | Количество записей на одной странице ответа |
| `resources` | array | Список ящиков |
| `total` | integer | Общее количество записей |
| `resources[].resourceId` | string (uint64) | Идентификатор почтового ящика |
| `resources[].count` | integer | Количество сотрудников, имеющих доступ к ящику |

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

У пользователя или приложения нет прав на доступ к ресурсу, запрос отклонен.

### 404 Not Found

Запрашиваемый ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса. Попробуйте повторно отправить запрос через некоторое время.
