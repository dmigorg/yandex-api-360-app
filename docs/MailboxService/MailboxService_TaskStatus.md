# Проверить статус задачи на изменение прав доступа

Возвращает статус задачи на управление правами доступа к делегированному ящику.

## Примечание

Чтобы выполнить запрос, приложению требуется разрешение на чтение информации о правах доступа к почтовым ящикам. Убедитесь, что вы включили его при настройке приложения на OAuth-сервере Яндекса.

## Request

```
GET https://api360.yandex.net/admin/v1/org/{orgId}/mailboxes/tasks/{taskId}
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `orgId` | integer | Идентификатор организации |
| `taskId` | string | Идентификатор задачи на управление правами доступа. Возвращается в ответе на запрос на изменение или удаление прав доступа к почтовому ящику |

## Responses

### 200 OK

Запрос успешно выполнен.

**Body:**

```json
{
  "status": "example"
}
```

| Параметр | Тип | Описание |
|----------|-----|---------|
| `status` | string | Статус выполнения задачи. Значения: `running` — выполняется; `complete` — успешно завершилась, права изменены; `error` — завершилась с ошибкой |

### 400 Bad Request

Некорректный запрос.

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

У пользователя или приложения нет прав на доступ к ресурсу, запрос отклонен.

### 404 Not Found

Запрашиваемый ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса. Попробуйте повторно отправить запрос через некоторое время.
