# Удалить общий ящик

Удаляет общий ящик из организации.

## Примечание

Чтобы выполнить запрос, приложению требуется разрешение на управление правами доступа к почтовым ящикам.

## Request

```
DELETE https://api360.yandex.net/admin/v1/org/{orgId}/mailboxes/shared/{resourceId}
```

### Параметры пути

| Параметр | Тип | Описание |
|----------|-----|---------|
| `orgId` | integer | Идентификатор организации |
| `resourceId` | string (uint64) | Идентификатор почтового ящика |

## Responses

### 200 OK

Запрос успешно выполнен.

**Body:** `{}`

### 400 Bad Request

Некорректный запрос.

```json
{
  "code": 0,
  "message": "example",
  "details": [
    {
      "@type": "example"
    }
  ]
}
```

### 401 Unauthorized

Пользователь не авторизован.

### 403 Forbidden

У пользователя или приложения нет прав на доступ к ресурсу.

### 404 Not Found

Запрашиваемый ресурс не найден.

### 500 Internal Server Error

Внутренняя ошибка сервиса. Попробуйте повторно отправить запрос через некоторое время.
