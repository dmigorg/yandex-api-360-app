# Безопасность: аудит-лог

Сервис для получения истории событий в организации.

## Разрешения на использование сервиса

- `ya360_security:audit_log_disk` — чтение событий аудит-лога Диска
- `ya360_security:audit_log_mail` — чтение событий аудит-лога Почты
