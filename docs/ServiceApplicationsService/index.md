# Сервисные приложения

## Обзор

Сервис для управления сервисными приложениями в Яндекс 360. Доступ ограничен только владельцами организации.

Перед использованием убедитесь, что ваш тарифный план поддерживает возможности управления сервисными приложениями.

## Разрешения на использование сервиса

- `ya360_security:service_applications_read` — просмотр списка сервисных приложений
- `ya360_security:service_applications_write` — полное управление, включая чтение и запись
